# SEO Keyword Strategy - Movie Database

## Primary Keywords (High Priority)
- `[movie-title] review` - High search volume
- `[movie-title] cast` - High search volume
- `watch [movie-title]` - High intent

## Secondary Keywords
- `[movie-title] release date`
- `[movie-title] rating`
- `movies like [movie-title]`

## Long-tail Keywords
- `is [movie-title] worth watching`
- `[movie-title] plot summary`
- `[actor-name] movies`

## Page Types We'll Create
1. **Movie Detail Pages** - Target: "[movie-title] review", "[movie-title] cast"
2. **Actor Profile Pages** - Target: "[actor-name] movies", "[actor-name] filmography"  
3. **Genre Pages** - Target: "best [genre] movies", "top rated [genre]"

## SEO Strategy
- Use movie titles in URLs (e.g., /movie/the-wrecking-crew)
- Include release year in meta descriptions
- Add ratings and cast in structured data
- Use high-quality movie posters as OG images
